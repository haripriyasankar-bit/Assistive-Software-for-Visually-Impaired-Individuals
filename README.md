# Assistive Software for Visually Impaired Individuals

## Overview
This project is an AI-powered assistive software designed to help visually impaired individuals navigate and interact with their surroundings more effectively. The system integrates **YOLO-based object detection**, **sensor technology**, and an **AI chatbot** to provide real-time environmental awareness, obstacle detection, and intelligent assistance.

The goal of this project is to improve accessibility, safety, and independence for visually impaired users through advanced computer vision and AI technologies.

---

## Features

### 🔹 Real-Time Object Detection
- Uses **YOLO (You Only Look Once)** for accurate and fast object detection.
- Identifies nearby objects in real time through a camera feed.
- Helps users recognize surroundings and important objects.

### 🔹 Sensor-Based Obstacle Detection
- Integrates sensors to detect nearby obstacles.
- Provides alerts for safer navigation.
- Enhances environmental awareness in indoor and outdoor spaces.

### 🔹 AI Chatbot Assistance
- Includes an **AI-powered chatbot** for user interaction.
- Provides voice/text-based responses and guidance.
- Assists users with queries and accessibility support.

### 🔹 Accessibility Support
- Designed specifically for visually impaired individuals.
- Improves navigation, object recognition, and independent movement.
